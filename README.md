
# 🚀 Mercado Ágil - Ultra-Enterprise SaaS v3.2

O **Mercado Ágil** é uma infraestrutura de comércio e serviços unificados de última geração, projetada para operar em escala global com performance local. Esta plataforma atende a múltiplas verticais de negócio através de uma arquitetura multi-tenant isolada e inteligência artificial nativa.

## 💎 Funcionalidades de Elite (150+ Recursos)

### 🏥 Verticais Especializadas (Interface Camaleão)
- **Saúde & Clínicas**: Prontuário Eletrônico (PEP) criptografado AES-256 e Telemedicina integrada.
- **Beleza & Estética**: Agenda operacional com disparos automáticos via WhatsApp Business.
- **Gastronomia**: Sistema KDS (Kitchen Display System), App de Garçom e Gestão de Mesas.
- **Varejo & Moda**: Controle de estoque multi-warehouse e rastreio de lotes via QR Code.
- **Pet Shop & Vet**: Monitoramento de internações, vacinas e banho & tosa.
- **Educação**: LMS Ágil Academy com suporte a 1.000+ cursos e manuais PDF.
- **Fitness & Studios**: Gestão de check-ins, mensalidades e Churn Rate IA.

### 🤖 Inteligência Artificial & Growth
- **Business Hub IA**: Consultoria de BI preditiva e geração de scripts de marketing.
- **Concierge IA**: Assistente virtual na vitrine do cliente para sugestões proativas.
- **AI Vision Scan**: Inventário volumétrico visual para controle de estoque rápido.
- **Creative Studio IA**: Geração de vídeos cinematográficos a partir de fotos estáticas (Veo 2.0).

### 💳 Finanças & Operações
- **PDV Cloud Pro**: Checkout unificado com emissão de recibos térmicos PDF.
- **Cashback Gold Tier**: Sistema de fidelidade por níveis (Bronze, Silver, Gold).
- **Logística & Frota**: Tracking de entregadores em tempo real.
- **Auditoria Master**: Painel de governança SaaS com monitoramento de infraestrutura global.

## 🛠️ Stack Tecnológica

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **UI/UX**: Shadcn/UI, Framer Motion, Lucide Icons.
- **Backend & DB**: Firebase Firestore (Real-time), Firebase Auth.
- **IA Engine**: Google Genkit com Gemini 2.5 Flash & Imagen 4.
- **Documentação**: jsPDF para geração de manuais técnicos pesados.

## 📦 Como Iniciar (Desenvolvimento)

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/mercado-agil.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente no `.env`:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=...
   GOOGLE_GENAI_API_KEY=...
   ```
4. Inicie o servidor:
   ```bash
   npm run dev
   ```

## 🌐 Deploy (Firebase App Hosting)
