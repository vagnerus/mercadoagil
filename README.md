# Mercado Ágil - Plataforma Enterprise SaaS & Academy

O **Mercado Ágil** é uma infraestrutura de comércio e serviços unificados de última geração, projetada para operar em escala global com performance local. Esta plataforma atende a múltiplas verticais de negócio (Beleza, Saúde, Varejo, Gastronomia, etc.) através de uma arquitetura multi-tenant isolada.

## 🚀 Funcionalidades de Elite

### 🏥 Verticais Especializadas
- **Saúde & Clínicas**: Prontuário Eletrônico (PEP) criptografado AES-256 e Telemedicina integrada.
- **Beleza & Estética**: Agenda operacional com disparos automáticos via WhatsApp Business.
- **Gastronomia**: Sistema KDS (Kitchen Display System), App de Garçom e Gestão de Mesas.
- **Varejo & Moda**: Controle de estoque multi-warehouse e rastreio de lotes via QR Code.

### 🎓 Ágil Academy Elite (LMS)
- **1.000 Cursos Profissionais**: Biblioteca massiva de especialização para lojistas e equipes.
- **Engine de PDF Pro**: Geração dinâmica de manuais de treinamento com 10+ páginas, sumário técnico e roteiros operacionais.
- **Player Imersivo**: Integração de vídeo em alta disponibilidade com conteúdo teórico denso.

### 🤖 Inteligência Artificial & Growth
- **IA Negotiator**: Otimização de preços baseada em Curva ABC e tendências de mercado.
- **Marketing Scripts**: Geração de campanhas de recuperação de carrinho e fidelização via IA.
- **AI Vision Scan**: Inventário volumétrico visual para controle de estoque rápido.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **UI/UX**: Shadcn/UI, Lucide Icons, Framer Motion.
- **Backend & Database**: Firebase Firestore (Real-time), Firebase Auth.
- **IA Engine**: Google Genkit com Gemini 2.5 Flash.
- **Documentação**: jsPDF para geração de manuais técnicos pesados.

## 📦 Como Iniciar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/mercado-agil.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente no `.env.local`:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=...
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
